# PDF增强处理功能 - 复杂表格、多列布局、页眉页脚

## 概述

在Docling基础上，新增了三个关键功能来解决PDF处理中的常见问题：

1. ✅ **页眉页脚自动过滤** - 去除重复的页眉页脚噪音
2. ✅ **跨页表格合并** - 自动合并分页的表格
3. ✅ **嵌套表格处理** - 扁平化复杂的嵌套表格
4. ✅ **多列布局支持** - 正确处理多栏文档（部分实现）

## 新增模块

### 1. 页眉页脚过滤 (`pdf_cleaner.py`)

**功能**：
- 自动检测在多页重复出现的文本（页眉、页脚、水印）
- 识别页码模式（"Page 1", "第1页", "- 1 -"等）
- 清理后返回纯净内容

**示例**：
```
原始PDF:
  Page 1: "公司机密 - 第1页\n实际内容\n版权所有"
  Page 2: "公司机密 - 第2页\n更多内容\n版权所有"
  Page 3: "公司机密 - 第3页\n其他内容\n版权所有"

处理后:
  Page 1: "实际内容"
  Page 2: "更多内容"
  Page 3: "其他内容"
```

**关键函数**：
- `detect_repeated_patterns()` - 检测重复模式
- `remove_page_numbers()` - 移除页码
- `clean_pdf_pages()` - 清理所有页面

---

### 2. 跨页表格合并 (`table_merger.py`)

**功能**：
- 检测表格是否在页面末尾被截断
- 识别下一页是否为表格延续
- 自动合并成完整表格

**示例**：
```
原始PDF:
  Page 5:
    | 产品 | 价格 |
    |------|------|
    | A    | 100  |
    | B    | 200  |
  
  Page 6:
    | C    | 150  |
    | D    | 250  |

处理后:
  Merged:
    | 产品 | 价格 |
    |------|------|
    | A    | 100  |
    | B    | 200  |
    | C    | 150  |
    | D    | 250  |
```

**关键函数**：
- `detect_incomplete_table()` - 检测不完整表格
- `is_table_continuation()` - 判断是否为表格延续
- `merge_cross_page_tables()` - 合并跨页表格

---

### 3. 嵌套表格处理 (`nested_table_handler.py`)

**功能**：
- 检测单元格内的嵌套表格
- 扁平化为单层结构
- 分割过宽的表格

**示例**：
```
原始（嵌套）:
  | 类别 | 详情 |
  |------|------|
  | 产品 | A | $100 | B | $200 |

处理后（扁平化）:
  | 类别 | 详情 |
  |------|------|
  | 产品 | A; $100; B; $200 |
```

**关键函数**：
- `detect_nested_table()` - 检测嵌套表格
- `flatten_nested_table()` - 扁平化
- `simplify_complex_table()` - 简化复杂表格

---

### 4. 多列布局处理 (`multicolumn_handler.py`)

**功能**：
- 检测多栏布局（2栏、3栏）
- 按正确顺序重排文本（左→右，上→下）

**示例**：
```
原始PDF布局:
  [左栏第1段]  [右栏第1段]
  [左栏第2段]  [右栏第2段]

错误提取: 左1 → 右1 → 左2 → 右2 ❌

正确提取: 左1 → 左2 → 右1 → 右2 ✅
```

**关键函数**：
- `detect_column_layout()` - 检测栏数
- `reorder_multicolumn_text()` - 重排文本

**注意**：此功能需要位置信息，当前为基础实现。

## 使用方法

### 配置

在 `.env` 文件中设置：

```bash
# PDF加载模式
PDF_LOADER_MODE=docling_enhanced  # 使用增强模式

# 可选：单独控制功能
PDF_ENABLE_CLEANING=true          # 启用页眉页脚清理
PDF_ENABLE_TABLE_MERGING=true     # 启用表格合并
```

### 四种模式对比

| 模式 | 功能 | 速度 | 适用场景 |
|------|------|------|---------|
| **pypdf** | 基础文本提取 | 最快 | 简单PDF |
| **docling** | Markdown表格 | 慢 | 标准表格PDF |
| **docling_enhanced** | Markdown + 清理 + 合并 | 最慢 | 复杂PDF（跨页表格、页眉页脚） |
| **hybrid** | Docling + OCR | 慢 | 表格 + 图片 |

### 测试

```bash
# 运行测试脚本
python scripts/test_enhanced_pdf.py

# 测试真实PDF
python scripts/test_enhanced_pdf.py data/docs/complex_report.pdf
```

测试会显示：
- ✅ 页眉页脚检测结果
- ✅ 跨页表格合并效果
- ✅ 嵌套表格处理结果
- ✅ 增强加载器与普通加载器对比
