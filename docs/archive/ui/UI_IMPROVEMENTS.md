## 🎨 设计系统

### 颜色方案

```css
/* 主色调 */
--accent: #1f7ae0;        /* 主要强调色 */
--ok: #0f9f63;            /* 成功/正确 */
--danger: #c83939;        /* 危险/错误 */
--muted: #65758d;         /* 次要文本 */

/* 渐变 */
linear-gradient(135deg, var(--accent), #00a782)
```

### 组件样式

**按钮：**
- `primary-action-btn` - 主要操作按钮（渐变蓝色）
- `secondary` - 次要按钮（边框样式）
- `link-button` - 链接样式按钮

**输入框：**
- `input-group` - 带标签的输入组
- `hint` - 提示文本（ok/error状态）
- `disabled-input` - 禁用状态

**卡片：**
- `auth-card` - 认证页面卡片
- `profile-card` - 个人资料卡片
- `info-box` - 信息提示框

## 📱 响应式设计

### 断点

- **桌面**: > 768px
- **平板**: 481px - 768px
- **手机**: ≤ 480px

### 移动端优化

**768px以下：**
- 双栏布局改为单栏
- 减小内边距
- 调整字体大小
- 按钮全宽显示

**480px以下：**
- 进一步缩小间距
- 优化触摸目标大小
- 简化导航元素
