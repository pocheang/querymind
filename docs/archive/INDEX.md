## 📌 Usage Guidelines

### When to Add to Archive

Move documents to archive when they:
- Describe completed work or resolved issues
- Are historical records of past decisions
- Contain version-specific information that's no longer current
- Are internal reports that have been superseded

### When to Keep in Active Docs

Keep documents in main docs when they:
- Describe current features or behavior
- Are actively maintained and updated
- Serve as user-facing documentation
- Are part of the current development workflow

## 🔍 Finding Archived Documents

Use the following command to search the archive:

```bash
# Search by keyword
grep -r "keyword" docs/archive/

# Search by date
find docs/archive/ -name "*2026-06*"

# List all version summaries
ls docs/archive/completion-reports/*v*.md

# List all version plans
ls docs/archive/plans/*v*.md

# List all summaries
ls docs/archive/summaries/

# List all plans
ls docs/archive/plans/
```
