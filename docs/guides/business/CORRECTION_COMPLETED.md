## 对齐的实际功能

### Agent Classes（代理类别）

文档现在正确反映了代码中的4种agent类别：

1. **cybersecurity** - 网络安全（攻击、漏洞、防护、应急）
2. **artificial_intelligence** - AI知识（大模型、RAG、提示词）
3. **pdf_text** - PDF处理（OCR、图片识别、文档提取）
4. **general** - 通用查询

### Skills（技能）

文档示例现在包含实际的技能类型：

**网络安全技能**：
- `cyber_attack_analysis` - 攻击分析
- `cyber_defense_hardening` - 防御加固
- `incident_response_playbook` - 应急响应

**通用技能**：
- `answer_with_citations` - 带引用答案
- `compare_entities` - 实体对比
- `pdf_text_reader` - PDF文本读取
- `ai_knowledge_assistant` - AI知识助手

### 路由逻辑

文档现在正确描述了路由决策过程：

```python
问题包含 "攻击"、"漏洞"、"防护"
→ agent_class = "cybersecurity"
→ skill = "cyber_attack_analysis" 或 "cyber_defense_hardening"

问题包含 "AI"、"机器学习"、"大模型"  
→ agent_class = "artificial_intelligence"
→ skill = "ai_knowledge_assistant"

问题包含 "PDF"、"OCR"、"图片"
→ agent_class = "pdf_text"
→ skill = "pdf_text_reader"
```
