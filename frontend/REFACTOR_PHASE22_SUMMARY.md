## 🎓 经验总结

### 何时提取工具函数和常量

**适合提取的情况：**
1. 函数在多个地方使用
2. 函数逻辑独立，不依赖组件状态
3. 常量在多个地方定义
4. 常量值较长，影响代码可读性

**不适合提取的情况：**
1. 函数只在一个地方使用
2. 函数强依赖组件状态
3. 常量只在一个地方使用
4. 提取后反而降低可读性

### 本阶段的关键决策

1. **选择提取而不是拆分组件** - 因为组件结构已经很清晰
2. **只做小范围优化** - 避免引入不必要的风险
3. **保持渐进式节奏** - 不追求一次性完美
4. **优先考虑可维护性** - 而不是过度优化
