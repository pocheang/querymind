## 使用方法

### 1. 文档索引

使用项目的文档索引功能将这些文档导入系统：

```bash
# 索引企业文档
python scripts/ingest_documents.py --path data/demo/enterprise/

# 索引技术文档
python scripts/ingest_documents.py --path data/demo/technical/
```

### 2. 运行评估

使用演示查询集进行系统评估：

```bash
python scripts/run_evaluation.py \
    --queries data/demo/evaluation/demo_queries.json \
    --systems vector_only,hybrid,rerank,full
```

### 3. 交互式测试

启动应用后，可以直接在界面中测试这些查询：

```bash
uvicorn app.api.main:app --reload
```

访问 http://localhost:8000/app 进行交互式测试。

## 评估指标

使用这些文档可以评估以下指标：

### 检索质量
- **Precision@5**：前5个结果的准确率
- **Recall@5**：前5个结果的召回率
- **MRR**：平均倒数排名
- **NDCG@5**：归一化折损累积增益

### 生成质量
- **Faithfulness**：答案忠实度
- **Answer Relevance**：答案相关性
- **Context Relevance**：上下文相关性

### 性能指标
- **响应时间**：P50, P95, P99
- **Token消耗**：输入/输出Token数
- **成本估算**：基于Token的成本
