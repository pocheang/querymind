# IT操作指南

## 1. VPN接入指南

### 1.1 VPN客户端安装

**Windows系统**：
1. 访问 https://vpn.company.com 下载客户端
2. 运行安装程序 `CompanyVPN_Setup.exe`
3. 使用默认安装路径：`C:\Program Files\CompanyVPN`
4. 安装完成后重启计算机

**macOS系统**：
1. 下载 `CompanyVPN.dmg`
2. 双击打开，拖拽到Applications文件夹
3. 首次运行需要授权系统扩展

### 1.2 VPN连接配置

1. 打开VPN客户端
2. 输入服务器地址：`vpn.company.com`
3. 输入域账号和密码
4. 选择"记住密码"（可选）
5. 点击"连接"

### 1.3 常见问题

**问题1：连接超时**
- 检查网络连接是否正常
- 确认防火墙未阻止VPN客户端
- 联系IT支持：it-support@company.com

**问题2：认证失败**
- 确认账号密码正确
- 检查账号是否被锁定
- 重置密码：https://password.company.com

## 3. 网络安全规范

### 3.1 密码安全

**密码要求**：
- 最少12个字符
- 包含大小写字母、数字、特殊字符
- 不能使用常见密码（如123456、password）
- 每90天强制修改一次

**密码管理**：
- 使用公司提供的密码管理器
- 不要在多个系统使用相同密码
- 不要将密码告诉他人
- 不要在纸上记录密码

### 3.2 邮件安全

**识别钓鱼邮件**：
- 检查发件人地址是否可疑
- 不要点击未知链接
- 不要下载可疑附件
- 遇到可疑邮件立即报告IT部门

**邮件使用规范**：
- 不要发送敏感信息（密码、信用卡号）
- 发送机密文件需加密
- 定期清理邮箱，避免超出配额
- 离职前导出个人邮件

### 3.3 数据安全

**数据分类**：
- **公开数据**：可以公开分享
- **内部数据**：仅限公司内部使用
- **机密数据**：需要特殊权限访问
- **绝密数据**：最高级别保护

**数据保护措施**：
- 机密文件存储在公司服务器
- 使用加密U盘传输敏感数据
- 不要将公司数据上传到个人云盘
- 定期备份重要数据
