# 贡献指南

感谢您对 QueryMind 的关注！

## 如何贡献

1. Fork 仓库
2. 创建分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启 Pull Request

## 代码规范

- Python: Black + Pylint + MyPy
- TypeScript: ESLint + Prettier
- 测试覆盖率 ≥ 80%

## 提交信息格式

- feat: 新功能
- fix: Bug修复
- docs: 文档更新
- test: 测试相关

感谢您的贡献！🎉
