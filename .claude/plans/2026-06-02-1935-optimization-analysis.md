## 🔍 发现的优化机会

### 1. 异常处理优化 🔴 高优先级

**发现**: 20个文件中存在裸`except:`语句或空`pass`

**问题**:
```python
# 不好的做法
try:
    something()
except:  # 捕获所有异常，包括KeyboardInterrupt
    pass  # 静默失败，难以调试
```

**建议优化**:
```python
# 好的做法
try:
    something()
except SpecificException as e:
    logger.error(f"Failed to do something: {e}")
    # 适当的错误处理或重新抛出
```

**影响的文件** (部分列表):
- `app/services/agent_execution_tracker.py`
- `app/services/query_guard.py`
- `app/graph/streaming/safe_wrappers.py`
- `app/services/prompt_checker.py`

---

### 3. 通配符导入 🟡 中优先级

**发现**: 多个文件使用`import *`

**问题**:
- 命名空间污染
- 依赖关系不明确
- IDE自动补全困难
- 潜在的名称冲突

**建议**: 使用显式导入
```python
# 替换
from module import *

# 改为
from module import specific_function, SpecificClass
```
